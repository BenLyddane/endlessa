Objective: Build a web application using the following technologies and libraries: - Next.js: A React framework for building server-side rendered and statically generated applications. - Shadcn components: A collection of UI components located in @/components/ui. - Tailwind CSS: A utility-first CSS framework for rapidly styling the application. - Shadcn styling: Utilizing Shadcn's foreground and background color system for consistent and visually appealing designs. - Supabase with Next.js: A backend service for handling data storage, retrieval, and real-time updates. The Supabase client can be created using the createClient function from @/utils/supabase/server for server components and @/uits/supabase/client for client components. The application should follow best practices for code organization, performance optimization, and maintainability. It should also be responsive and work well on various devices.
